## 第1章 起步

Python 是一门高级解释型语言。目前有 v2 和 v3 两个版本，但是 [CPython 核心开发小组已计划在 2020 年 4 月发布 v2.7 版本后停止更新 v2 系列][5]，v3 将成为唯一的推荐版本。这意味着以后将不用纠结使用哪个版本。

```py
print("Hello World!")
```

如果要获取 Python 解释器的完整路径：

```sh
$ type -a python3
```
